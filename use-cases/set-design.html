Set / Production Design — keep art & logistics in sync

Organize locations, build dressing lists, and sync needs with scenes and shots.

Key tabs

  • Locations DB (photos, access, power, noise, contacts)
  • Props & Dressing (status, owner, storage, continuity notes)
  • Shot Needs (pulls items per scene/shot)

How it flows

  1. Tag scenes with location + time of day → stripboard groups them.
  2. Shot list references Props/Locations → auto-pull needs.
  3. Call sheets include location info and priority props.
